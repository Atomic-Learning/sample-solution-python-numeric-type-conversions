Let's practice converting between numeric types in Python, step by step.

# Start with a float my_number = 2.6 print(my_number)

Convert the float to an integer using int() (truncates toward zero).

round_down = int(my_number) print(round_down)

Round to the nearest integer using round().

round_nearest = round(my_number) print(round_nearest)

Convert the rounded integer back to a float.

new_float = float(round_nearest) print(new_float)

Now let's try the same steps with a negative number.

my_number = -3.6 print(my_number) round_down = int(my_number) print(round_down) round_nearest = round(my_number) print(round_nearest) new_float = float(round_nearest) print(new_float)
